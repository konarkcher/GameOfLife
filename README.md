# GameOfLife

## Threads

### Commands available:

* START \<thread_count> \<source.csv>
* START \<thread_count> RANDOM \<height> \<width>
* STATUS
* RUN \<iteration_count>
* STOP
* QUIT
* END — same as QUIT but also stop the whole program

Other commands may cause undefined behaviour.
